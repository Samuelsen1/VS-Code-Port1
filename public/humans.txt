/* ACCESSIBILITY */

This website is designed to be accessible to everyone, including:
- Screen reader users
- Keyboard-only navigation
- Low vision users (high contrast, text scaling)
- Color blind users (non-color dependent UI)
- Users with dyslexia (specialized fonts available)
- Users with motion sensitivity (animation control)

Features:
- 11 accessibility options with 3-level intensity control
- Skip links for keyboard navigation
- ARIA labels on all interactive elements
- Semantic HTML structure
- Alt text on all images
- Bilingual support (EN/DE)
