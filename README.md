# React Portfolio Website

This is a React-based portfolio website showcasing instructional design and e-learning development work. It features a responsive design using Tailwind CSS and includes sections for projects, skills, experience, and contact information.

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm start
```

## Built With

- React
- Tailwind CSS
- Lucide React Icons
