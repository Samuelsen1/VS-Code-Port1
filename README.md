# React Portfolio Website

This is a React-based portfolio website showcasing technical writing and digital learning design work. It features a responsive design using Tailwind CSS and includes sections for projects, skills, experience, and contact information.

The portfolio highlights expertise in:
- **Technical Writing**: API documentation, user guides, knowledge bases, DITA XML
- **Digital Learning Design**: E-learning modules, instructional design (ADDIE), SCORM content
- **Accessibility**: WCAG 2.1 compliance, inclusive design, multilingual content
- **Content Localization**: AI-assisted translation, cross-cultural adaptation

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm start
```

## Built With

- React
- Tailwind CSS
